SERP Intelligence Tool for Living Systems Counselling (livingsystems.ca), a small Bowen Family Systems Theory nonprofit in North Vancouver, BC. Maps Problem-Aware search queries to Solution-Aware content using Google SERP data (via SerpAPI) + Anthropic LLM reports.

**Why:** Small nonprofit needs to compete for organic search against large directories and medical-model content. Uses differentiated positioning based on Bowen Theory.

**How to apply:** All LLM prompts inject client context from `config.yml → analysis_report`. Recommendations should fit a small nonprofit budget and avoid mismatched audiences (e.g., court-mandated reunification services).
